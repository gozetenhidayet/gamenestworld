SEO STEP 5 - INDEXING READY PACKAGE

This package skips Step 4 category pages and prepares the site for indexing.

Included:
- Updated robots.txt
- Updated sitemap.xml
- Strong robots meta tags on HTML pages
- all-urls-submit-to-google.txt
- priority-urls-submit-first.txt
- google-indexing-url-list.csv
- Turkish Google Search Console guide

Upload every file and folder in this package to your GitHub Pages site root.
Then submit sitemap.xml in Google Search Console.
